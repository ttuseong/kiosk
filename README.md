# 기능 소개
## 메인
![메인 페이지 설명](https://user-images.githubusercontent.com/38309884/95945393-56563080-0e25-11eb-94f2-4ffb32cace56.png)
① 로그인
로그인 버튼을 클릭할 경우 로그인 페이지로 이동

② 로그인 페이지
관리자의 아이디와 페스워드를 정확하게 입력한 경우 로그인 가능

③ 로그인 후 변경 사항
로그인에 성공할 경우 관리자 로그인 버튼이 관리자 페이지와 로그아웃 버튼으로 변경된다

④ 연습할 키오스크 선택
연습할 키오스크를 선택하는 것으로 현재 구현된 KFC와 롯데리아 키오스크 연습이 가능하다

## 롯데리아
![롯데리아](https://user-images.githubusercontent.com/38309884/95969543-d5f5f680-0e49-11eb-8305-88391e25d397.png)
① 광고 페이지
- 롯데리아 키오스크에 처음 들어가면 보이는 화면으로 광고를 클릭하면 다음 페이지로 이동 가능
 - 사용자가 다음에 어떤 행동을 해야할지 모르는 경우를 대비하여 일정시간 동안 클릭 이벤트가 없을 경우 오른쪽 위에 툴팁을 통해 다음에 해야할 행동에 대해 설명을 줌

② 결제 방법 선택
- 사용자가 원하는 결제 방법을 클릭하면 다음 페이지로 이동 가능

③ 메인 화면
- 카테고리를 클릭하여 원하는 카테고리로 이동 가능
- 메뉴를 클릭하여 주문 가능 만약 버거인 경우 ④으로 이동 
- 주문 목록, 전체 수량, 총 금액을 확인할 수 있고 동일 제품의 갯수를 늘릴 수 있다.
- 취소하기 버튼을 클릭할 경우 결제 방법을 선택하는 페이지로 이동
- 결제하기 버튼을 통해 결제 페이지로 이동 가능

④ 햄버거 단위 선택
- 버거만 버튼을 클릭할 경우 단품이 주문됨
- 세트를 클릭할 경우 ⑤으로 이동
- 엑스 버튼을 클릭할 경우 주문없이 ③으로 이동

⑤ 사이드 변경
- 카테고리를 클릭하여 원하는 카테고리로 이동 가능
- 디저트를 클릭한 경우 자동으로 음료로 이동
- 취소 버튼을 클릭할 경우 주문없이 ③으로 이동
- 선택완료 버튼을 클릭한 경우 현재 클릭된 정보를 토대로 주문

⑥ 결제 페이지
- 사용자가 주문한 내역과 총 금액을 확인할 수 있음
- 포장이나 매장 식사를 선택
- 취소하기를 클릭할 경우 메인 화면으로 이동
- 모든 선택을 한 후 주문완료 버튼을 클릭하면 주문이 완료되어 키오스크를 선택하는 페이지로 이동

## KFC
![KFC](https://user-images.githubusercontent.com/38309884/95969604-e8703000-0e49-11eb-88bf-fbdcbcdc2ba2.png)
① 메인 페이지
- 카테고리를 클릭하여 원하는 카테고리로 이동 가능
- 메뉴를 클릭하여 주문 가능
- 전체 삭제 버튼을 통해 모든 주문 내역을 지울 수 있음
- 주문된 메뉴의 경우 수량 조절 가능
- 주문된 메뉴의 수와 총 금액 확인 가능
- 메뉴 목록이 비어있을 경우 주문 버튼클릭했을 때 다음으로 못 넘어감

② 단위 선택
- 햄버거나 치킨을 클릭했을 경우 나옴
- 사용자는 원하는 단위를 클릭후 확인 버튼 클릭
- 치킨의 경우 사이드의 변경이 없기 때문에 바로 주문되어 메인 화면으로 이동

③ 변경 가능한 내용 확인
- 사용자가 선택한 단위에 따라 변경 가능한 내용이 다름
- 상단에 있는 플러스, 마이너스 버튼을 통해 수량 조절 가능
- 현재 선택된 메뉴의 금액 확인 가능
- 확인버튼을 클릭할 경우 주문 목록에 내용이 추가되고 메인 화면으로 이동

④ 토핑 변경
- 조절하고 싶은 토핑의 플러스, 마이너스 버튼을 클릭하여 토핑 수 조절

⑤ 사이드 변경
- 원하는 사이드로 변경
- 선택된 사이드에 따라 추가 금액 적용

⑥ 장소 선택
- 매장 또는 포장 선택

⑦ 추천 메뉴
- 결제 전에 사용자에게 추가적으로 메뉴 추천
- 중복 선택 불가능

⑧ 주문 목록 확인
- 메인 페이지에서 주문한 내용과 추천 메뉴에서 선택한 내용들이 모두 들어간 주문 목록
- 최종 결제 금액 확인 가능

⑨ 결제 수단 선택
- KFC에서 제공하는 결제 수단을 확인 후 선택

⑪ 결제 수단 및 결제 금액 확인
- 사용자가 선택한 결제 수단과 총 결제 금액을 확인

⑫ 결제 확인 
- 실제 키오스크에서는 결제가 이루어지는 화면이지만, 연습 사이트이기 때문에 확인 버튼을 통해 넘어갈 수 있음

## 관리자 페이지 - 메뉴
![메뉴](https://user-images.githubusercontent.com/38309884/95969665-fa51d300-0e49-11eb-97e9-64b548d0230e.png)
① 메인 화면
- 이미지 업로드할 때 드래그앤 드롭으로 가능
- 카테고리 선택 후 메뉴에서 기존 메뉴를 선택한 경우 해당 매뉴 수정 가능
- 토핑 정보 부분에서 사용되는 토핑 추가 및 변경 가능

② 프로모션/할인 선택
- 여러 제품을 묶은 프로모션인지 가격 할인 행사인지 선택

③ 프로모션 구성
- 묶음 상품을 만들 때 어떤 제품들을 묶을 지 선택
- 플러스 버튼을 통해 수를 늘릴 수 있음
- 마이너스 버튼을 통해 선택했던 제품을 지울 수 있음

④ 할인 적용
- 할인가 또는 할인율을 입력하여 적용

⑤ 단위
- 햄버거의 세트가 기본으로 감자튀김하고 콜라가 기본 구성품으로 있는 것 처럼 공통적으로 기본 구성품을 넣어주기 위해 사용
- 추가/삭제/수정 가능

⑥ 단위 추가/수정
- 이름 변경, 구성품 변경 가능
- 플러스 버튼을 통해 구성품목을 늘릴 수 있음
-  마이너스 버튼을 통해 구성품목을 줄일 수 있음

⑦ 관련 메뉴 설정
- 단위와 달리 기본 구성품이 없고, 같은 이름의 메뉴를 올릴 때 사용
ex) 양념치킨 3조각, 양념치킨 5조각을 만들려고 할 경우 양념치킨을 관련 메뉴로 설정

## 관리자 페이지 - 토핑
![토핑](https://user-images.githubusercontent.com/38309884/95969730-0fc6fd00-0e4a-11eb-828d-36714489d60a.png)
① 메인 페이지
- 새로운 토핑 추가 가능
- 토핑 이름을 클릭한 경우 해당 토핑 수정 가능
- 토핑 이름을 통한 검색 가능
- 토핑 다섯개를 기준으로 페이징

② 사용법 툴팁
- 물음표부분에 마우스를 올릴 경우 사용법에 대한 툴팁 제공

③ 이미지 미리보기
- 사진 미리보기란에 있는 아이콘에 마우스를 올릴 경우 해당 토핑에 적용되는 이미지 미리보기 가능

## 관리자 페이지 - 카테고리
![카테고리](https://user-images.githubusercontent.com/38309884/95969779-1ce3ec00-0e4a-11eb-984d-6f8451f9fbc4.png)
① 메인 페이지
- 새로운 카테고리 추가 가능
- 카테고리에 이미지를 넣어야하는 경우 이미지 넣기 체크 박스를 클릭 후 이미지 추가
- 카테고리 이름을 클릭한 경우 해당 카테고리 수정 가능
- 카테고리 이름을 통한 검색 가능
- 카테고리 다섯개를 기준으로 페이징

② 사용법 툴팁
- 물음표부분에 마우스를 올릴 경우 사용법에 대한 툴팁 제공

③ 이미지 미리보기
- 사진 미리보기란에 있는 아이콘에 마우스를 올릴 경우 해당 카테고리에 적용되는 이미지 미리보기 가능
## 관리자 페이지 - 통계
![통계](https://user-images.githubusercontent.com/38309884/95969809-25d4bd80-0e4a-11eb-9c7e-ec31fc548337.png)
① 월별 성공률
 - 해당 년도의 월별 키오스크 주문 성공률을 확인 가능
 
② 전체 성공률
 - 모든 년도에 대한 키오스크 주문 성공률
 
③ 사용자가 포기하는 구간
- 사용자가 중간에 포기하고 나간 지점을 기록

④ 사용자가 투표한 키오스크 난이도 순위
- 주문에 성공한 사용자들 키오스크 난이도가 쉽다고 선택한 비율로 매장의 순위를 매김
